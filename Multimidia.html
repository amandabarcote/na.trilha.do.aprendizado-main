Tag Vídeo

A tag video permite que um vídeo seja exposto no site, um vídeo com controles e ainda com vídeos “reservas” caso algum vídeo esteja indisponível ou não possa ser aberto. Veja um exemplo abaixo:

A Tag video pode ter o “poster” que serve para adicionar uma imagem estática, é a imagem que fica aparecendo enquanto o vídeo não é inicializado pode ter o “controls” que serve para adicionar os controles padrões da mídia pode ter também o “height” para a altura e “width” para a largura.
Entre as Tags video tem o “source” que serve para adicionar um vídeo, ele vai tentando inicializar o vídeo de cima para baixo, caso não tenha uma resposta passa a tentar inicializar o próximo, o “src” dentro da tag é para indicar o caminho e o “type” é para indicar o tipo do arquivo.

Tag Imagem (picture)

A tag "picture" serve para anexar uma imagem, e pode ser colocadas condições para a troca de imagem baseado no tamanho da tela, ou seja, se a tela diminuir ou aumentar o tamanho é possível mudar a imagem para outra de sua preferencia. Veja um exemplo abaixo:

A tag picture pode ter entre ela as tags “source” e “img”, a tag img é destinada a ser executada caso nenhuma condição prevista seja satisfeita, caso contrario a tag source que coincidir com a situação atual vai ser executada, dentro dela tem a “media” que serve para colocar a condição e a “srcset” que é o local onde se coloca o caminho da imagem a ser exposta caso o media seja satisfeito.
