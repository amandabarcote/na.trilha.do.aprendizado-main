Table

Essa tag é responsável por desenvolver uma tabela, a estrutura do código é composta pelo (table, tr, th e td). O table é a tag que vai definir o inicio e o fim da tabela, o tr delimita linha da tabela, dentro do tr pode se ter varias colunas mas apenas uma linha, o th define o titulo da tabela e o td o conteúdo da tabela.Também exitem as tags caption e colspan, a caption define um título para a tabela e o colspan define quantos espaços um título ou informação tomara na tabela.

Exemplo:

X X X
